Overview

In this project I explored the Gapminder dataset to understand relationships between GDP, life expectancy, and population across countries and years.

Libraries Used

Library Purpose
pandas Data manipulation
plotly.express Interactive visualisations

Analysis

import pandas as pd
import plotly.express as px

# Load built-in Gapminder dataset
df = px.data.gapminder()

# Filter for most recent year
df_2007 = df[df['year'] == 2007]

# Interactive scatter plot
fig = px.scatter(
    df_2007,
    x="gdpPercap",
    y="lifeExp",
    size="pop",
    color="continent",
    hover_name="country",
    log_x=True,
    title="GDP per Capita vs Life Expectancy (2007)",
    labels={"gdpPercap": "GDP per Capita", "lifeExp": "Life Expectancy"}
)
fig.show()

Key Findings

Note

Countries with higher GDP per capita generally show higher life expectancy, though with diminishing returns at very high income levels.

  • Europe and Oceania consistently show the highest life expectancy.
  • Africa shows the most variation, suggesting other factors at play.
